# 💰 Sistema de Folha de Pagamento - C#

Aplicação Console desenvolvida para praticar lógica de programação e estruturação de métodos em C#.

---

## 📌 Funcionalidades

- Entrada do nome do funcionário
- Entrada do salário bruto
- Cálculo automático de:
  - INSS
  - IRRF
- Exibição do salário líquido
- Opção para novo cálculo

---

## 🛠️ Tecnologias Utilizadas

- C#
- .NET Console Application

---

## 🧠 Conceitos Aplicados

- Estrutura de repetição (do/while)
- Métodos estáticos
- Estruturas condicionais
- Manipulação de decimal
- Formatação monetária
